# Sales and Revenue Analysis Dashboard

## ABOUT THE PROJECT
This project presents an interactive Power BI dashboard designed to analyze sales performance and revenue across different regions and time periods.
The dashboard provides a clear overview of total revenue, sales distribution by state and city, and monthly trends over multiple years. It allows users to easily identify top-performing locations, monitor growth patterns, and explore variations in sales over time.

Key features include:

1.Total revenue overview  
2.Sales breakdown by state and city  
3.Monthly revenue trends by year  
4.Interactive filters for city and year selection

This analysis helps uncover valuable business insights, such as regional performance differences, top revenue generating cities, and overall sales evolution.

## Data Processing
The dataset was cleaned and transformed using Power Query to ensure accurate and consistent analysis.
